Field Tile
==========

A field tile is a very simple tile: it just displays a field of its context honoring the widget customizations present in the schema tagged values (but it doesn't take into account the other tagged values such as when to display something or not, as this is reserved for the )

    >>> from plone.testing.z2 import Browser
    >>> app = layer['app']
    >>> browser = Browser(app)
    >>> browser.handleErrors = False
    >>> content = layer['portal']['test_type']
    >>> browser.open(content.absolute_url())
    >>> browser.open(content.absolute_url() + '/@@plone.app.standardtiles.field?field=contributors')
    >>> 'id="form-widgets-contributors"' in browser.contents
    True
