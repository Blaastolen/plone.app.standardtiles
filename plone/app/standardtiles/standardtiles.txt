First, we set up a browser instance::

    >>> from plone.testing.z2 import Browser
    >>> app = layer['app']
    >>> browser = Browser(app)
    >>> browser.handleErrors = False

Transient tiles can be accessed directly using a querystring syntax::

    >>> portal = layer['portal']
    >>> browser.open(portal.absolute_url() + '/@@plone.app.standardtiles.colophon/tile1')

Persistent tiles require a bit more work, exercising data managers::

    >>> video_tile = portal.restrictedTraverse('@@plone.app.standardtiles.video/tile1')
    >>> from plone.tiles.interfaces import ITileDataManager
    >>> dataManager = ITileDataManager(video_tile)
    >>> dataManager.set({'youtubeURL': 'http://www.youtube.com/watch?v=ayPKvFNz8aE'})
    >>> print video_tile()
    <html><body><object width="425" height="344"><param name="movie" value="http://www.youtube.com/v/ayPKvFNz8aE&hl=en_GB&fs=1&"></param><param name="allowFullScreen" value="true"></param><param name="allowscriptaccess" value="always"></param><embed src="http://www.youtube.com/v/ayPKvFNz8aE&hl=en_GB&fs=1&" type="application/x-shockwave-flash" allowscriptaccess="always" allowfullscreen="true" width="425" height="344"></embed></object></body></html>

To test the portlet manager tile, we can use the mock portlet manager we registered in the testing layer::

    >>> browser.open(portal.absolute_url() + '/@@plone.app.standardtiles.portletmanager/tile1?manager=mock.portletmanager')
    >>> print browser.contents
    <html><body>Portlet Manager Renderer output.</body></html>
